# Resources

This directory contains UI resources for the MAEVN VST3 plugin.

## Contents

Place the following resources here:

- **Icons/** - Plugin icons and button graphics
- **Skins/** - UI theme configurations
- **Fonts/** - Custom fonts for the plugin interface
- **Images/** - Background images and decorative elements

## Default Theme

The plugin uses a default dark theme. Custom themes can be added as JSON configuration files in the `Skins/` subdirectory.

## Icon Specifications

- Format: PNG with transparency
- Recommended sizes: 16x16, 24x24, 32x32, 48x48
- Color space: sRGB
